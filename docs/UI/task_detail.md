# タスク詳細画面 (task_detail_view.dart)

## 概要

タスクカードのタップから遷移する画面。  
Rockのロック中タスクは**閲覧専用**（完了操作不可）で、  
アクティブなRockとPebble/Sandは完了・編集が可能。

---

## コアコンセプト：Rockのロック状態を明示

- **Rockアクティブ（現在番号）**: フル操作可能
- **Rockロック中**: グレーアウト＋「前のタスクを完了してから進めます」バナー表示
- **Pebble / Sand**: 常にフル操作可能

---

## 表示要素

### 1. ヘッダー
- 戻るボタン（左上）
- 編集ボタン（右上・鉛筆アイコン）
- 「…」メニュー（削除・カテゴリ変更）

### 2. ロック中バナー（Rockロック中のみ表示）
- 「🔒 #N 番のタスクを完了してから進められます」
- 薄い背景色で画面上部に表示
- 完了・チェックボタンはすべて非活性

### 3. タイトルエリア
- カテゴリバッジ（🔒 Rock #N / 🪨 Pebble / 🏖️ Sand）
- タスクタイトル（大きめフォント）
- 完了チェックボックス（ロック中は非活性・グレーアウト）

### 4. メタ情報エリア
- 期限日（設定ありの場合・期限切れは赤字）
- 作成日

### 5. サブタスクエリア（サブタスクありの場合のみ）
- サブタスク一覧・各チェックボックス
- ロック中Rockはサブタスクも非活性
- サブタスク完了数 / 全件数＋プログレスバー

### 6. メモエリア（メモありの場合のみ）
- メモ全文

### 7. 完了ボタン（アクティブタスクのみ）
- 画面下部固定「✓ 完了にする」ボタン
- Rockの場合: 完了後に「次のタスク（#N+1）がアンロックされました！」トースト

---

## インタラクション

| 操作 | 動作 |
|---|---|
| 「完了にする」タップ（Rock） | 完了→次のRockアンロック→ホームへ戻る |
| 「完了にする」タップ（Pebble/Sand） | 完了→ホームへ戻る |
| ロック中タスクのチェックをタップ | 「前のタスクを完了してください」トースト |
| カテゴリ変更（メニュー） | Rockに変更→末尾に追加、Rock→Pebble等→順番リセット |

---

## 状態管理（task_detail_view_model.dart）

| 状態 | 型 | 説明 |
|---|---|---|
| `task` | Task? | 表示中タスク |
| `isLocked` | bool | Rockのロック中フラグ |
| `isLoading` | bool | データ取得中 |
| `errorMessage` | String? | エラーメッセージ |

---

## 使用するRepository

| 依存先 | 用途 |
|---|---|
| `TaskRepository` | タスク取得・完了更新・Rock順番管理 |

---

## エラー・ローディングの扱い

- ロック中タスクへの操作: トーストで案内
- データ取得失敗: エラー表示＋リトライ
- 削除失敗: エラースナックバー
